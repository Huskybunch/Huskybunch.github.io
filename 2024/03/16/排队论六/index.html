Introduction to Poisson Process

Stochastic(uncertain/random) process随机过程

  • general definition
    • family of random variables that describes the evolution(through time) of a process
  • notation
    • X(t), t∈T is the state of the process at index t
      • T is often interpred as time常被解释为时间
  • comment
    • when T is a countable set, stochastic process is discrete-time
    • when T is an interval of the real line, stochastic process is continuous-time
  • examples
    • size of population at time t ->t = 2013 -> discrete time
    • distance a particle moves in time t -> t = in 2 hrs ->continuous time
  • Today we will focus on an elementary stochastic process初等随机过程 known as Poisson process

Poission process

  • time interal:contimuous-time stochastic process
  • monitoring the occurrence of some event of interst
    • arrivals->arrivals of customers, reeived calls -> in a morning shift
  • average rate(λ) at which the event occurs does not change over time
  • randon variable
    • N(t) ≡ number of times the event occurs in (0, t)

Poisson process implications泊松过程含义

  • N(0) = 0

  • independent increments

    • what happens in an interval of time does not depend on any other non-overlapping interval of time
    • what happens in [1,3] does not impact what happens in[4, 6]

  • N(t + s) - N(s) ≡ Poisson RB with mean λt = N(t)：number of occurances in intervals if length time t

    • does not depend on s
    • does depend on t

Poisson random variable

  • example
    • the number of customers that arrive to a bank in one hour -> t = 1 -> N(1)
  • terminology
    • X (N(t)) is a poisson random variable with mean α (α = λt)
  • Notation
    • X ~ Poisson(α)
    • N(t) = Poisson(λt)

X ~ Poisson(α)

  • range

    • X ∈{0,1，……}
    • X is a discrete random variable

  • PMF

    • is 3 times happened in 60 minutes

  • CDF

  • expected value E(X) = λt

  • variance V(X) = λt
