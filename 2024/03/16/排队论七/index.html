Introduction to Exponential Random Variable指数分布随机变量

Arrival time

  • definition
    • time of event occurrence
    • called”arrival time” instead of occurrence time” because of the use of Poisson to model customer arrival使用泊松模型来模拟客户到达
  • notation
    • S(j) ≡ arrival time of customer
    • S(2): arrival time of the 2nd customer

Interarrival time到达间隔时间

  • definition

    • time between event occurrences/arrivals

  • notation

    • T(j) ≡ interarrival time of customer j

  • relationship to arrival times

    • T(1) ≡ S(1)
    • T(2) ≡ S(2) - S(1)
    • T(j) ≡ S(j) - S(j - 1) j = 3,4,…

  • Statement

    • T(j) is an exponential random variable having rate λ（value of j is not relevant)
    • λ does not change with j , same as poisson

  • interpretation解释

    • if the number of arrivals**(N(t))** is Poisson, then the time between arrivals**(interarrival time)** is exponential

Exponential random variable

  • terminology
    • X is an exponential random variable with rate λ
  • notation
    • X ~ expon(λ)
    • in Poisson process: T(j)(interarrival time) ~expo(λ)
  • range
    • X∈ (0,∞)
    • X i s a contimuous random variable
